# CloudPage - Authentic Design

A modern, responsive website for CloudPage with an elegant dark theme and image gallery.

## Features

- **Modern Navigation**: Icon-based navigation bar with Home, Videos, Services, About, Contact, and External links
- **Image Gallery**: Interactive gallery with main display and thumbnail navigation
- **Responsive Design**: Fully responsive layout that works on all devices
- **Dark Theme**: Elegant dark overlay design with background images
- **Multiple Pages**: Complete website with dedicated pages for each section

## Pages

1. **Home (index.html)**: Main landing page with image gallery
2. **Videos (videos.html)**: Video gallery showcasing your content
3. **Services (services.html)**: Display of all cloud services offered
4. **About (about.html)**: Company information and mission
5. **Contact (contact.html)**: Contact form and information with map

## File Structure

```
├── index.html              # Main homepage

│   ├── videos-style.css    # Videos page styles
│   ├── services-style.css  # Services page styles

│   └── contact-style.css   # Contact page styles
├── js/
│   ├── authentic-script.js # Main JavaScript
│   ├── jquery-3.4.1.min.js
│   └── jquery.magnific-popup.min.js
├── img/                    # Images folder
├── video/                  # Videos folder
├── fontawesome/            # Font Awesome icons
└── assets/                 # Logo and assets
```

## Color Scheme

- Primary: #ffcc00 (Yellow/Gold)
- Background: Dark with rgba overlays
- Text: White with various opacity levels
- Accent: Transparent overlays

## Customization

### Changing Images
- Replace images in the `img/gallery/` folder
- Update image sources in `index.html`

### Updating Content
- Edit text directly in HTML files
- Modify styles in respective CSS files
